Project Description

This project implements the TOPSIS (Technique for Order of Preference by Similarity to Ideal Solution) method.

The application:

Accepts a CSV file

Takes weights and impacts as input

Calculates TOPSIS score

Generates ranking

Sends the result file via email

Is deployed as a live web application

Is published as a Python package on PyPI
